## 总：模块概要（给人阅读）

本模块是项目的集中配置中心。硬件接线、舵机动作、人员检测、门控时间、BLE 标识和网络行为等固定参数都从这里提供，其他模块不再各自保存一份相同数值。

把参数集中管理的意义，是让“系统行为为什么发生变化”更容易追踪。例如门关得太快，可以从门控时间配置开始检查；更换传感器接线，可以从 I²C 引脚配置开始修改，而不必在多个源文件中搜索数字。

### 配置怎样影响系统

```text
配置参数
├─ 硬件层：I²C 引脚、舵机引脚、TOF 地址
├─ 动作层：开关门角度、舵机速度
├─ 判断层：距离阈值、防抖、驻留和关门延时
├─ 通信层：BLE UUID、设备名、Web 端口、mDNS
└─ 运行层：串口、调试输出、连接与重启时间
```

### 参数分组

| 分组 | 主要内容 | 主要使用者 |
|---|---|---|
| `Pins` | I²C 和舵机引脚 | 系统入口、硬件驱动 |
| `Servo` | 开关门角度、运动间隔和步长 | 舵机控制、门控 |
| `Tof` | 地址、检测阈值和有效量程 | TOF、门控 |
| `Door` | 防抖、人员驻留和关门延时 | 自动门控制 |
| `Ble` | 设备名、服务和特征 UUID | BLE 配网 |
| `Network` | mDNS、端口和网络时序 | Wi-Fi、Web 服务 |
| `Debug`、`Serial`、`System` | 日志、串口和重启时序 | 系统入口及各模块 |

### 修改前先判断影响范围

| 影响等级 | 典型参数 | 修改后要做什么 |
|---|---|---|
| 硬件安全 | 舵机角度、步长、引脚 | 核对接线，脱离门体进行台架测试 |
| 业务行为 | 检测阈值、驻留、关门延时 | 重新验证人员靠近、离开和折返场景 |
| 外部兼容 | BLE UUID、设备名、mDNS、端口 | 同步手机端、网页入口或其他客户端 |
| 调试体验 | 日志开关、打印间隔 | 检查串口负载和日志内容 |

这些配置目前都是编译期常量，修改后需要重新编译并烧录。配置模块只定义数值和公共状态类型，不执行硬件控制或网络连接；具体参数怎样被使用，由对应功能模块决定。

## 分：代码重建规格（给 AI 或修改代码的开发者阅读）

### 1. 文件和依赖

`src/config/Config.h` 使用 `#pragma once` 并包含 `Arduino.h`。文件首先定义全局 `enum class DoorState { Closed, Open, WaitingToClose };`，随后定义 `namespace Config`。

### 2. 常量完整表

| 命名空间 | 名称 | 类型 | 值 |
|---|---|---|---|
| Pins | i2cSda | `uint8_t` | 21 |
| Pins | i2cScl | `uint8_t` | 22 |
| Pins | servo | `uint8_t` | 5 |
| Servo | openAngle | `int` | 90 |
| Servo | closedAngle | `int` | 0 |
| Servo | updateIntervalMs | `unsigned long` | 15 |
| Servo | openStep | `int` | 30 |
| Servo | closeStep | `int` | 10 |
| Tof | address | `uint8_t` | 0x29 |
| Tof | changeThresholdCm | `float` | 2.5F |
| Tof | maxValidDistanceMm | `uint16_t` | 800 |
| Door | presenceTimeoutMs | `unsigned long` | 1500 |
| Door | closeDelayMs | `unsigned long` | 2000 |
| Door | detectionDebounceMs | `unsigned long` | 200 |
| Ble | deviceName | `char[]` | `AutoDoor` |
| Ble | serviceUuid | `char[]` | `a550d001-0000-a550-d001-a550d0010001` |
| Ble | wifiScanCharacteristicUuid | `char[]` | `a550d001-0000-a550-d001-a550d001a001` |

| Serial | startupDelayMs | `unsigned long` | 500 |

全部成员使用 `constexpr`。字符串使用 `constexpr char name[]`。

### 3. 不变量与验收

- 时间字段后缀 `Ms` 表示毫秒，距离字段后缀 `Cm` 表示厘米。
- UUID、设备名和 mDNS 名称属于外部契约，修改需同步 BLE/Wi-Fi 文档。
- 重建时不得把 `DoorState` 移入 `Config` 命名空间，否则现有调用会失配。
