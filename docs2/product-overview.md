# 产品与范围

## 产品目标

AutoDoorBLE 是本地运行的智能自动门控制器。ESP32 根据距离变化自动驱动舵机，并通过 BLE 完成 Wi-Fi 配网，通过局域网 Web 页面提供状态、手动控制和重新标定。

## 当前能力

- HC-SR04 距离测量、有效值过滤、三点中值滤波和启动标定。
- AUTO 模式的接近检测、防抖、驻留和延迟关门。
- MANUAL 模式通过网页设置舵机目标角度。
- 舵机非阻塞渐进运动。
- BLE 扫描附近 Wi-Fi、提交凭据和推送连接状态。
- Wi-Fi 凭据持久化、断线检测、定时重连和 mDNS。
- Web 控制台和 REST API。

## 系统边界

- 控制和状态查看默认只在局域网内提供。
- BLE 当前只承担 Wi-Fi 配网，不作为日常门控协议。
- 当前没有认证、TLS、OTA、MQTT、实时曲线或 WebSocket。
- 当前只支持一个 HC-SR04 和一个舵机。

## 控制优先级

MANUAL 高于 AUTO。进入 MANUAL 后自动门状态机暂停，舵机更新继续执行；切回 AUTO 后恢复自动检测。

## 质量目标

- 主循环保持非阻塞，各模块通过 `update()` 增量推进。
- 网络断开不阻止本地自动门逻辑。
- 无效距离数据不直接触发状态转换。
- 未来功能保留无需云服务的基本自动门能力。
