# Web 运行时参数配置

- 状态：Proposed
- 关联路线图：Phase 6

## 目标

通过 Web 修改可安全调整的自动门参数并持久化到 NVS。

## 非目标

- 不允许修改引脚、BLE UUID、串口或启动期硬件常量。
- 本阶段不实现认证；实施前必须评估未授权修改风险。

## 目标行为

- `GET /api/config` 返回公开可修改参数。
- `POST /api/config` 校验并原子更新允许参数。
- 重启后保留有效配置；损坏数据回退到默认值。
- 每个参数定义类型、单位、最小值、最大值和默认值。

## 待决策

- 单项还是整份更新。
- NVS schema 版本和迁移。
- 即时生效还是安全状态生效。
- 未认证时是否允许修改机械风险参数。

## 验收条件

- [ ] 越界值、错误类型和未知 key 被拒绝且不改变配置。
- [ ] 有效配置重启后仍存在。
- [ ] NVS 损坏时使用安全默认值启动。
- [ ] API 不暴露凭据。
- [ ] 配置变更不阻塞主循环。
