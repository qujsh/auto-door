# 硬件与安全

## 硬件和引脚

| 元件/信号 | 当前配置 |
|---|---|
| 主控 | ESP32 |
| 距离传感器 | HC-SR04，TRIG GPIO 2，ECHO GPIO 1 |
| 舵机 | SG90/MG996R，PWM GPIO 5 |
| 舵机电源 | 独立 5V，与 ESP32 共地 |

## 安全约束

- 舵机必须使用容量合适的独立电源并共地。
- HC-SR04 ECHO 为 5V 时必须确认输入容限；需要时使用分压或电平转换。
- 修改角度、步长或延时前确认机械行程，避免堵转。
- 上电默认目标保持关闭角度，除非规格给出安全迁移方案。
- 关门变化必须覆盖等待期间再次检测到人的场景。
- 运动变化先脱离门体负载预检，再进行受控硬件测试。

参数数值的权威定义在 `Config.h`，本文不重复完整参数表，避免漂移。
